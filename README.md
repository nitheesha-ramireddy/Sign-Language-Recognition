## 🤟 Sign Language Recognition System

## 📌 Overview
This project converts sign language gestures into text and speech using deep learning, MediaPipe, and NLP.

---

## 🚀 Features
- Hand landmark detection using MediaPipe
- Alphabet recognition using neural networks
- Video-to-text conversion
- NLP-based word correction
- Text-to-speech output

---

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- MediaPipe
- OpenCV
- TextBlob (NLP)
- gTTS (Speech)

---

## 🔄 Workflow
Video → Frames → Hand Landmarks → Prediction → Word Formation → NLP Correction → Speech
